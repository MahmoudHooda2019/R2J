# R2J

**R2J** is a tool to convert an `R.txt` file into an `R.java` file.

## Download

- [JAR File](https://github.com/MahmoudHooda2019/R2J/raw/main/out/R2J.jar)
- [EXE File](https://github.com/MahmoudHooda2019/R2J/raw/main/out/R2J.exe)

## How To Use

1. **Launch the Application**  
   <img src="https://github.com/MahmoudHooda2019/R2J/blob/main/images/1.png?raw=true" alt="Step 1" width="600"/>

--------------
2. **Drag and Drop Your `R.txt` File**  
   <img src="https://github.com/MahmoudHooda2019/R2J/blob/main/images/2.png?raw=true" alt="Step 2" width="600"/>

   _or_

   **Click the "Browse" Button and Choose the `R.txt` Path**  
   <img src="https://github.com/MahmoudHooda2019/R2J/blob/main/images/3.png?raw=true" alt="Step 3" width="600"/>

--------------
3. **Set the Package Name for Your Java Class**  
   <img src="https://github.com/MahmoudHooda2019/R2J/blob/main/images/4.png?raw=true" alt="Step 4" width="600"/>

--------------
4. **Click the "Convert" Button**  
   <img src="https://github.com/MahmoudHooda2019/R2J/blob/main/images/5.png?raw=true" alt="Step 5" width="600"/>

--------------
5. **Find the Output `R.java` File**  
   - The output file will be saved in the same directory as the `R.txt` file.  
   <img src="https://github.com/MahmoudHooda2019/R2J/blob/main/images/6.png?raw=true" alt="Step 6" width="600"/>
